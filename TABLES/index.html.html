Web Application Development

Courese Code:ITM 1131

Course Level:Year One, Semester One

Credit Unit:4

Course Description

This course introduces students to Web technologies-HTML,CSS,JavaScript;CSSframeworks;Javascript frameworks.Students will learn to implement modern web pages using HTML and CSS framework.The course also introduces students to full stack web application programming using JavaScript

Course Objectives

  1. Describe web technologies(HTML,CSS &JavaScript).CSS frameworks,and JavaScript frameworks
  2. Implement a simple but modern web page with HTML and CSS.
  3. build a simple but functional web application using JavaScript

Learning Outcomes

  • Use web technologies(HTML,CSS $JavaScript),CSS frameworks,and JavaScript frameworks
  • Design and create a simple but functional web paage using HTML and CSS framework,
  • Compose and build a simple full stack web application using Javascript framework
